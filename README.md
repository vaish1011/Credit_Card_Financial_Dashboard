# Credit_Card_Financial_Dashboard
Power Bi Dashboard

Project Overview: This project is an interactive Power BI dashboard designed to analyze Credit Card Transactions and Customer Behaviour.
It provides key insights into revenue trends, transaction patterns, customer demographics, and card usage to support data-driven decision-making.

Objectives
- Analyze overall credit card revenue and transaction volume
- Identify top-performing customer segments
- Understand spending behavior by category and demographics
- Track quarterly and weekly revenue trends
- Compare performance across card categories and payment methods

Dashboards Included
1) Credit Card Transaction Report
2) Credit Card Customer Report

Tools & Technologies
- Power BI Desktop
- DAX for calculated measures
- Data Modeling
- Data Visualization & Dashboard Design

- Blue card holders contribute the highest revenue
- Swipe transactions dominate over online and chip usage
